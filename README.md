# AI Content Generator - Personalized Content Creation Platform

## Overview
An intelligent web application that generates high-quality, tailored content using Groq API and Llama 3.3 LLM.

## Features
- ✅ 4 Content Types: LinkedIn Posts, Ad Copy, Professional Emails, Conversational Text
- ✅ Real-time content generation
- ✅ User-friendly Streamlit interface
- ✅ Cloud-based LLM integration
- ✅ Secure API key management

## Tech Stack
- **Frontend**: Streamlit
- **Backend**: Groq API
- **Model**: Llama 3.3 70B
- **Language**: Python 3.x

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
