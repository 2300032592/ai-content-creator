## 3. Professional Email

### Characteristics
- **Platform**: Corporate communication
- **Audience**: Colleagues, clients, partners
- **Purpose**: Convey information, request action, maintain relationships
- **Tone**: Formal, respectful, clear

### Content Specifications

| Aspect | Details |
|---|---|
| **Length** | 100-250 words (brief to moderate) |
| **Tone** | Formal, professional, courteous |
| **Structure** | Greeting ‚Üí Purpose ‚Üí Details ‚Üí CTA ‚Üí Closing |
| **Grammar** | Perfect, formal language |
| **Clarity** | Direct, no ambiguity |
| **Formality** | High (Dear Sir/Madam) |
| **Closing** | Professional sign-off |

### Optimized System Prompt
```

## 4. Conversational Text

### Characteristics
- **Platform**: Chat, social media, customer service
- **Audience**: General audience, customers
- **Purpose**: Build rapport, engage, inform casually
- **Tone**: Friendly, natural, relatable

### Content Specifications

| Aspect | Details |
|---|---|
| **Length** | Variable (typically 50-200 words) |
| **Tone** | Casual, friendly, conversational |
| **Language** | Contractions, informal words |
| **Structure** | Natural flow, not rigid |
| **Engagement** | Questions, personal touches |
| **Emojis** | Appropriate, not overdone |
| **Personality** | Relatable, authentic |

### Optimized System Prompt
```
"Generate friendly conversational text that:
- Uses natural, casual language
- Includes contractions (don't, it's, etc.)
- Feels like talking to a friend
- Engages the reader with questions or hooks
- Is authentic and relatable
- Can include appropriate emojis
- Maintains engaging flow"

## 7. Optimization Recommendations

### For LinkedIn Posts
1. Add data/statistics for credibility
2. Include personal experience stories
3. Tag relevant professionals
4. Post during peak hours (Tuesday-Thursday)

### For Ad Copy
1. A/B test different CTAs
2. Emphasize unique benefits
3. Use power words (transform, unleash, breakthrough)
4. Add social proof (testimonials, numbers)

### For Professional Email
1. Use recipient's name (personalization)
2. Keep paragraphs short
3. Use bullet points for clarity
4. Add signature with contact info

### For Conversational Text
1. Start with a question or hook
2. Use active voice
3. Keep sentences short
4. End with a conversation starter
